# Pal-Bench

Pal-Bench is a public benchmark for AI-generated **Pal** artwork.

You can explore submissions, vote on quality, and compare model performance over time.

## What you can do

- **Rate Pals** in a fast, swipe-like flow
- **Browse submissions** by model and search by name
- **View leaderboards** for top-performing outputs
- **See platform stats** such as coverage and hallucination trends

## Why it exists

Pal-Bench helps compare model output quality in a simple, transparent way:
- same benchmark space,
- community voting,
- clear model-level results.

## Built with

- React + Vite
- Convex
- Tailwind CSS
